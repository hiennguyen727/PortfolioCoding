Portfolio

From my class's inception to the present day, my coding projects have evolved from basic exercises to sophisticated applications, showcasing my ongoing growth and dedication to mastering new technologies.

City of Refuge Website

Github

Collaborating with fellow developers at a Hackathon event hosted by the City of Refuge, we engaged in an intensive 8-hour sprint to build a fully functional website for the organization. The website's purpose was to serve as a mentor-mentee portal.

Hien.Tv

Github

Hien.Tv is a movie website that allows users to search for their favorite movie! It utilizes an API with over 200,000 movies to allow users to find and click on a movie of their choice, and get basic information on the movie.

Finish Up

Github

Finish Up is a simple web-based to do list that allows users to stay organized in their everyday tasks. It features add,updating, and deleting tasks, making it a versatile tool for daily task management.

Time Yourself

Github

Time Yourself is a versatile stopwatch with a minimalist design that allows precise tracking of hours, minutes, seconds, and milliseconds. Its straightforward and efficient interface offers users a dependable way to measure elapsed time for various activities and purposes.

Dazzle Duo!

Github

Dazzle Duo! is a fun and interactive website for couples to figure out what to do. It features randomly generated dinner and wine recipes, personality type quiz, and date ideas varying with the quiz results.

Peppa Plug

Github

Peppa Plug is a fun and informative website containing various information on hot peppers. It features a gallery containing many peppers varying in scoville levels and allows users to favorite their peppers to their accounts.
